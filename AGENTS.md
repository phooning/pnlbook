# AGENTS

This project is a minimalist persistent PnL calendar with basic watchlists.

Prefer `ast-bro` for large file reads.

## General commands

```bash
bun i
bun dev
```

```bash
bun run build
```

```bash
bun run test
```

```bash
bun lint
bun format
bun check
```

## shadcn instructions

Use the latest version of Shadcn to install new components, like this command to add a button component:

```bash
pnpm dlx shadcn@latest add button
```
